### 🐟 Game 18 — 어항에 물고기를 넣어줘

**`Ally8_fish.js`**

#### 개요

여러 개의 어항이 나타나고, **물고기가 3마리 들어 있는 어항을 찾아 터치**하는 수량 선택 게임이다.

#### 교육 요소

- 수 감각 (3 세기)
- 시각 변별력

#### 게임플레이 메커니즘

| 항목 | 내용 |
|------|------|
| 조작 | 어항 이미지 터치 |
| 성공 조건 | 물고기 3마리가 있는 어항 터치 |
| 실패 조건 | 오답 선택 또는 타임아웃 |
| 난이도 | 어항 3개 (각 1~5마리), 물고기 종류·색상·배치 랜덤 |

#### 화면 연출

| 화면 | 성공 시 | 실패 시 |
|------|---------|---------|
| 모바일 | 선택 어항 + 숫자 이미지 표시 | 흔들림 + 숫자 표시 |
| TV | 물고기 3마리 헤엄 애니메이션 + 물방울 이펙트 | 물고기 놀란 표정 |

### 🎨 Game 19 — 같은 색 풍선을 찾아봐

**`Ally9_balloon.js`**

화면 상단에 **기준 색상 풍선**이 표시되고, 하단에 여러 색 풍선 중 **같은 색상의 풍선을 터치**하는 색상 매칭 게임이다.

#### 교육 요소

- 색상 인식 및 분류
- 시각 변별력

#### 게임플레이 메커니즘

| 항목 | 내용 |
|------|------|
| 조작 | 풍선 이미지 터치 |
| 성공 조건 | 기준과 같은 색상 풍선 터치 |
| 실패 조건 | 다른 색상 풍선 터치 또는 타임아웃 |
| 난이도 | 기준 색상 랜덤 (5색), 선택지 풍선 4개 |

### 🐘 Game 21 — 큰 동물을 찾아보자

**`Ally11_big_animal.js`**

#### 개요

3마리 동물이 표시되고, 그 중 **가장 큰(실제 크기 기준) 동물을 터치**하는 크기 비교 게임이다. 화면 속 이미지 크기는 같게 표시하여 아이의 실제 지식을 테스트한다.

#### 교육 요소

| 실패 조건 | 오답 선택 또는 타임아웃 |

## 공통 적용 사항

- 모든 신규 게임은 기존 게임과 동일한 **5초 타이머** 적용
- **TV/모바일 이중 렌더링** 구조 유지 (`renderTV()` 메서드 구현)
- 공통 인터페이스: `init()`, `start()`, `onButtonDown()`, `onTimeout()`, `renderTV()`, `cleanup()`
- **성공/실패 오디오** 기존 시스템 재사용 (`audioManager`)
- 결과 화면은 기존 오버레이 방식 동일 적용 (홈/재시도 버튼)
- 에셋 폴더 구조: `assets/Ally{N}_{name}/` 하위에 `mobile/`, `tv/` 분리
